**`SOC-Midterm`**

This repository contains the midterm project report, sample output images from object detection tasks (such as pedestrian detection), object-tracking videos, trained model weights, and evaluation metrics.

🟢 **Showcase Results:** Explore the **[Outputs](./Outputs)** folder to view detection images and tracking videos across a diverse range of pedestrian scenes and benchmark sequences.

🟣 **Training Insights:** Browse the **[yolo_runs](./yolo_runs)** folder to examine training curves, model weights, evaluation metrics, experiment configurations, and other artifacts generated during the YOLOv5 training process.
